# 🚀 Futuristik Platformanı Görmək Üçün Təlimat

## Server işləyir! İndi bunları edin:

### 1. **Brauzeri Açın və Keçin:**
```
http://localhost:8000
```

### 2. **Əgər Köhnə Versiyanı Görürsünüzsə:**

#### A) **Hard Refresh Edin (Cache Təmizləyin):**
- **Windows:** `Ctrl + Shift + R` və ya `Ctrl + F5`
- **Firefox:** `Ctrl + Shift + R`
- **Chrome:** `Ctrl + Shift + R`

#### B) **Developer Tools ilə Cache Təmizləyin:**
1. `F12` basın (Developer Tools açmaq üçün)
2. Network tab-a keçin
3. "Disable cache" qutusunu işarələyin
4. Səhifəni yeniləyin

### 3. **Görəcəyiniz Yeni Futuristik Effektlər:**

✨ **Loading Screen:**
- "INITIALIZING SYSTEMS..." yazısı ilə yükləmə ekranı

🌐 **Arxa Fon Effektləri:**
- Matrix Rain (0 və 1 rəqəmləri yağışı)
- Cyber Grid (hərəkətli şəbəkə)
- Data Streams (şaquli məlumat axınları)

🛡️ **Hero Section:**
- Fırlanan Cybersecurity Shield (3 qatlı qalxan)
- Holographic Server Rack
- Yanıb-sönən LED işıqlar
- Terminal tipli başlıqlar

💫 **Bütün Bölmələrdə:**
- Neon yaşıl (#00ff88) və cyan (#00ccff) parıltılar
- Holographic kart effektləri
- Section keçid animasiyaları
- Hover effektləri

👥 **Müştərilər Bölməsi:**
- customers.jpg şəkli böyük ölçüdə göstərilir
- Holographic çərçivə effekti

### 4. **Əgər Hələ də Görmürsünüzsə:**

```powershell
# Serveri dayandırın (Ctrl+C)
# Sonra yenidən başladın:
python manage.py runserver --insecure
```

### 5. **Browser Console-da Xətaları Yoxlayın:**
- `F12` → Console tab
- Qırmızı xəta varsa, mənə bildirin
